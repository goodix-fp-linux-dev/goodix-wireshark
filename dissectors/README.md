# Dissectors

Dissectors plugins for [Wireshark](https://www.wireshark.org/).

## Usage

To enable dissecting in Wireshark copy one of the following `.lua` script into `~/.local/lib/wireshark/plugins`. Then restart Wireshark.

## List

+ [`51x0`](51x0.lua)
+ [`51x7`](51x7.lua)
+ [`52xd`](52xd.lua)
+ [`53x5`](53x5.lua)
+ [`53xd`](53xd.lua)
+ [`55x4`](55x4.lua)
